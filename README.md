# Telecomunicaciones-Megaline
Proyecto 5. La empresa Megaline quiere saber que plan tarifario genera más ingresos. El análisis de dos planes tarifarios diferentes, con las librerias de Pandas y Numpy en Python permitió conocer cual de ellos proporcionaba mejores ingresos, así como las diferencias en diferentes zonas geogreaficas, llevando a un mejor manejo de presupuesto de publicidad.
